# Week 1: React Fundamentals

Welcome to Week 1 of the React Development Curriculum! This week lays the foundation for your React journey by covering the essential concepts that every React developer must master.

## 📚 What You'll Learn This Week

### Core Topics Covered:
1. **What is React** - Understanding React's role in modern web development
2. **JSX Fundamentals** - Learning React's syntax extension
3. **Components** - Building blocks of React applications
4. **State Management Basics** - Managing component state with hooks
5. **Event Handling** - Making your components interactive

## 🗂️ Study Guide Structure

### [01. What is React](./01-What-is-React-README.md)
**Duration:** Day 1-2  
**Key Concepts:**
- React library vs framework
- Virtual DOM and its benefits
- Component-based architecture
- Setting up development environment

**Real-world Connection:** Understanding why companies like Facebook, Netflix, and Airbnb chose React

### [02. JSX Fundamentals](./02-JSX-Fundamentals-README.md)
**Duration:** Day 2-3  
**Key Concepts:**
- JSX syntax and rules
- Expressions in JSX
- Conditional rendering
- JSX fragments

**Real-world Connection:** Building dynamic user interfaces like social media feeds

### [03. Components](./03-Components-README.md)
**Duration:** Day 3-4  
**Key Concepts:**
- Function vs Class components
- Props and prop types
- Component composition
- Children prop

**Real-world Connection:** Creating reusable UI components for e-commerce sites

### [04. State Management Basics](./04-State-Management-Basics-README.md)
**Duration:** Day 4-5  
**Key Concepts:**
- useState hook
- State vs props
- State immutability
- Multiple state variables

**Real-world Connection:** Building interactive forms and user profiles

### [05. Event Handling](./05-Event-Handling-README.md)
**Duration:** Day 5-7  
**Key Concepts:**
- SyntheticEvents
- Event handlers
- Form handling
- Preventing default behavior

**Real-world Connection:** Creating interactive dashboards and user input forms

## 🎯 Learning Objectives

By the end of Week 1, you will be able to:

✅ **Understand React Fundamentals**
- Explain what React is and why it's useful
- Set up a React development environment
- Understand the Virtual DOM concept

✅ **Write JSX Code**
- Create JSX elements with proper syntax
- Use JavaScript expressions in JSX
- Implement conditional rendering

✅ **Build React Components**
- Create functional components
- Pass and use props effectively
- Compose components together

✅ **Manage Component State**
- Use the useState hook
- Update state immutably
- Handle multiple state variables

✅ **Handle User Interactions**
- Attach event handlers to elements
- Handle form submissions
- Prevent default browser behaviors
